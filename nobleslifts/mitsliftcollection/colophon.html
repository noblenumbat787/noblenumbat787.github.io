Behind the scenes

Mitsubishi Lift Collection is proud to have chosen GitHub Pages as its platform of choice for hosting this site.

This site is written in pure HTML and CSS using Visual Studio Code.
If this site appears a bit too plain- you are not wrong. It's design is deliberately kept simple to allow for compatibility with older browsers as well as slower internet connections - less bloat means less loading time after all; plus that also allows for easier maintenance as there are less moving parts.

For all the privacy nerds out there; rejoice - this site does not track you in any form, and does not use any sort of browser cookies.

Browser compatibility

This site will work on any browser that supports basic HTML and CSS. (yes, that includes Internet Explorer!)
A text-mode browser like Lynx, will also work well, thanks to the simplicity of this site (though you will not be able to see any images).
