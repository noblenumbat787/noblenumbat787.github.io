1970s

Elepet

The Elepet was Mitsubishi's first design of lift; having been introduced in 1960.
Originally based on relay-controlled logic; Elepet lifts soon became fully computerised with microprocessor-based logic with the Elepet Advance, released in 1979.

A set of Mitsubishi Elepet lifts in duplex.

Two former Mitsubishi Elepet lifts; originally in a duplex set.

This particular set of lifts has been modernised; the right lift has been modded with an ELEMOTION (see more about it here), and the left one with a generic lift.
Thus, this set of lifts are no longer a duplex set; reason being for this is the fact that each of the lifts had been modernised at different times.
